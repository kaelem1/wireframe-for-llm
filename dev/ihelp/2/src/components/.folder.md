# Folder: /Users/kaelem/dev/ihelp/2/src/components

1. **地位**：编辑器与预览界面组件层

2. **逻辑**：左栏组件面板仅保留项目名与等宽名称网格，组件按钮持续选中直到改选或再次点击取消；`BoardCanvas` 负责空白提示层、连续放置、框选多选、任意组件移动缩放、对齐辅助线、fit 缩放与右键菜单；右栏承接英文交互编辑、多选批量操作与带目标态反馈的图层拖拽排序，预览层复用块级交互并保持 fit。

3. **约束**：不引入外部 UI 库；所有块用 HTML 绝对定位渲染。

## 成员清单

- `Toolbar.tsx`：顶部工具栏，仅保留设备信息、导入导出与预览入口
- `ComponentPalette.tsx`：左侧组件面板，提供项目名入口与名称化组件网格
- `BoardCanvas.tsx`：中央线框画布，负责点击/拖拽放置、框选多选、移动缩放与 fit 缩放
- `BoardCanvas.test.tsx`：统一模式、点击放置、拖拽定尺寸、框选多选、图层拖拽、fit 缩放与预览交互回归测试
- `InteractionPanel.tsx`：右侧交互与图层面板，提供英文文案、多选批量操作与图层拖拽反馈
- `BoardStrip.tsx`：底部画板条，支持画板重排、删除与整板创建副本
- `PreviewOverlay.tsx`：预览模式，块级 tap/longPress/swipe 与 fit 预览
- `SetupDialog.tsx`：新建项目弹层
- `SettingsDialog.tsx`：未接入主路径的设置与左右对比还原测试弹层
- `AIGeneratorDialog.tsx`：未接入主路径的 AI 生成弹层
- `WireframeBlock.tsx`：通用线框块渲染，预览态由根节点承接交互
