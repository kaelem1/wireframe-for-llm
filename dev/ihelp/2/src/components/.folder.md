1. **地位**：编辑器与预览界面组件层

2. **逻辑**：左栏组件面板仅保留项目名与按浏览器语言自动出文案的等宽名称网格，并将“通用块”独立置顶显示；palette header 已删除，桌面端由内层单滚动承接，移动端切回页面滚动；toolbar 与 preview 特性已删除；`BoardCanvas` 支持 Option/Alt 拖拽复制、越界移动缩放与单句 empty state，并在待放置时把图层块切到纯展示态，拖动/缩放只做本地预览并在松手时一次性写入历史；`App` 复用现有复制链路承接 Cmd/Ctrl 复制粘贴，并在待放置态底部居中展示退出放置 toast；右栏顶部承接 2/3 主按钮导出 + 1/3 次按钮复制 JSON，下方是图层与交互面板；图层与画板保持自动聚焦与拖拽反馈，重复点击已选图层保持选中不抖动，图层项仅显示主名称；弹窗能力从组件类型收敛为交互描述。

3. **约束**：不引入外部 UI 库；所有块用 HTML 绝对定位渲染。

## 成员清单

- `ComponentPalette.tsx`：左侧组件面板，提供项目名入口、名称化组件网格，并将通用块独立置顶
- `BoardCanvas.tsx`：中央线框画布，负责点击/拖拽放置、框选多选、Option/Alt 拖拽复制、移动缩放与 fit 缩放
- `BoardCanvas.test.tsx`：统一模式、点击放置、拖拽定尺寸、框选多选、图层拖拽、fit 缩放、placement toast 与一次性 undo 回归测试
- `InteractionPanel.tsx`：右侧交互与图层面板，统一标题结构，顶部提供主导出/次复制 JSON 双按钮，并承接多选批量操作、图层拖拽反馈、自动聚焦与弹窗描述配置
- `BoardStrip.tsx`：底部画板条，支持画板重排、删除、整板创建副本、自动聚焦，并将菜单弹层提升到根层避免裁切
- `SetupDialog.tsx`：新建项目弹层，按浏览器 locale 自动出文案
- `SettingsDialog.tsx`：未接入主路径的设置与左右对比还原测试弹层，主要文案已双语化
- `AIGeneratorDialog.tsx`：未接入主路径的 AI 生成弹层，主要文案已双语化
- `WireframeBlock.tsx`：通用线框块渲染，预览态由根节点承接交互，并提供更明显的选中强调样式钩子；待放置时可切到纯展示态
