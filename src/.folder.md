# Folder: /Users/kaelem/dev/wireframe-for-llm/src

1. **地位**：前端应用源码根目录

2. **逻辑**：`main.tsx` 挂载应用；`App.tsx` 组装按浏览器语言自动出文案的画布+右栏编辑器壳层、可点击的待放置 toast、复制/导出 JSON 成功短时 toast、底部画板条与快捷键；`stores/` 管待放置组件、多选、画板复制、组件复制粘贴、图层命名防重与语言状态，并在待放置期锁住其他图层选中入口，项目名编辑允许临时空值；`components/` 渲染居中的新建弹层、画布顶部居中悬浮组件工具栏、左上角项目名输入、contain 居中完整显示且无 stage 外壳的可越界编辑画布、左下角 100% 默认缩放控件与右栏顶部 60px 导出/复制 JSON/GitHub 入口，新建弹层只保留固定设备预设，画板菜单脱离横向滚动列表裁切；`types/` 与 `utils/` 维护组件注册表、双语词典与导入导出能力；`*.test.tsx` 为开发回归文件，不进入 app 生产构建。

3. **约束**：只保留编辑器必需代码；视觉与数据结构必须对应导出 JSON。

## 成员清单

- `components/`：编辑器界面与统一线框画布
- `stores/`：Zustand 状态、撤销重做与多选命令
- `types/`：结构化 JSON 类型与组件目录分组类型
- `utils/`：常量、完整组件注册表、AI、导入导出、contain 缩放与布局辅助
- `App.tsx`：应用壳与画布+右栏主工作区布局
- `index.css`：全局样式、setup 居中、中心区无外层 padding、顶部居中悬浮组件工具栏、画布 contain 居中、左下角缩放控件且 stage 无外壳视觉
- `main.tsx`：挂载入口
