# Folder: /Users/kaelem/dev/wireframe-for-llm/src

1. **地位**：前端应用源码根目录

2. **逻辑**：`main.tsx` 挂载应用；`App.tsx` 组装按浏览器语言自动出文案的三栏编辑器壳层、可点击的待放置 toast、复制 JSON 成功短时 toast、底部画板条与快捷键；`stores/` 管待放置组件、多选、画板复制、组件复制粘贴、图层命名防重与语言状态，并在待放置期锁住其他图层选中入口，`components/` 渲染左栏项目名+组件库、中间可越界编辑的画布与右栏顶部导出/复制 JSON/GitHub 入口及下方图层/交互，右栏交互区独立滚动、图层区保持常驻可见；`types/` 与 `utils/` 维护组件注册表、双语词典与导入导出能力；`*.test.tsx` 为开发回归文件，不进入 app 生产构建。

3. **约束**：只保留编辑器必需代码；视觉与数据结构必须对应导出 JSON。

## 成员清单

- `components/`：编辑器界面与统一线框画布
- `stores/`：Zustand 状态、撤销重做与多选命令
- `types/`：结构化 JSON 类型与组件目录分组类型
- `utils/`：常量、完整组件注册表、AI、导入导出与布局辅助
- `App.tsx`：应用壳与主工作区布局
- `index.css`：全局样式与工作区两侧栏高度、滚动分配
- `main.tsx`：挂载入口
