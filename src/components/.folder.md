# Folder: /Users/kaelem/dev/ihelp/2/src/components

1. **地位**：编辑器与预览界面组件层

2. **逻辑**：组件面板点击后进入待放置；`BoardCanvas` 负责 wireframe 空白提示层、单击默认放置、拖拽定尺寸放置、选中移动、八向缩放、尺寸提示、对齐辅助线、fit 缩放与右键菜单；底部画板条处理命名与切换，预览层复用块级交互并保持 fit。

3. **约束**：不引入外部 UI 库；所有块用 HTML 绝对定位渲染。

## 成员清单

- `Toolbar.tsx`：顶部工具栏，仅保留项目、导入导出与预览入口
- `ComponentPalette.tsx`：左侧组件面板，点击后进入待放置状态并在侧栏内独立滚动
- `BoardCanvas.tsx`：中央 wireframe 画布，负责点击/拖拽放置、移动缩放、空白提示层与 fit 缩放
- `BoardCanvas.test.tsx`：wireframe 点击放置、拖拽定尺寸、侧栏滚动、fit 缩放、预览交互与画板重名回归测试
- `InteractionPanel.tsx`：右侧交互与图层面板，图层区承接剩余高度并独立滚动
- `BoardStrip.tsx`：底部画板条，依赖 store 拦截重名
- `PreviewOverlay.tsx`：预览模式，块级 tap/longPress/swipe 与 fit 预览
- `SetupDialog.tsx`：新建项目弹层
- `SettingsDialog.tsx`：未接入主路径的设置与左右对比还原测试弹层
- `AIGeneratorDialog.tsx`：未接入主路径的 AI 生成弹层
- `WireframeBlock.tsx`：通用线框块渲染，预览态由根节点承接交互
