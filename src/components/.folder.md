## 成员清单

- `ComponentPalette.tsx`：左侧组件面板，提供项目名输入与七类活跃组件网格
- `BoardCanvas.tsx`：中央线框画布，负责 contain 居中完整显示、stage 无外壳视觉、固定尺寸 empty state、点击/拖拽放置、框选多选、Option/Alt 拖拽复制、移动缩放、连续放置锁定高亮与缩放
- `PreviewOverlay.tsx`：旧预览覆盖层，保留在源码树中但不进入生产构建
- `BoardCanvas.test.tsx`：统一模式、七类组件面板、项目名清空、完整显示缩放、stage 无外壳视觉、固定尺寸 empty state、点击放置、拖拽定尺寸、连续放置锁定高亮、框选多选、图层名草稿编辑、图层拖拽与 grip 提示、导出 JSON 备注、placement/copy/export toast、GitHub logo 按钮、固定设备预设与一次性 undo 回归测试；仅供开发回归，不进 app 生产构建
- `InteractionPanel.tsx`：右侧交互与图层面板，统一标题结构，顶部提供 60px 高浅色导出/复制 JSON/GitHub logo 三按钮，并在复制/导出成功时通知上层显示 toast，同时承接多选批量操作、图层名草稿提交、图层拖拽反馈与 grip 提示、自动聚焦、描述/属性编辑与弹窗描述配置
- `BoardStrip.tsx`：底部画板条，支持画板重排、删除、整板创建副本、自动聚焦，并将菜单弹层提升到 `board-strip` 直系层避免裁切，按钮文案跟随 locale
- `SetupDialog.tsx`：新建项目弹层，只展示固定设备预设，标题、项目名与创建按钮跟随 locale
- `SettingsDialog.tsx`：未接入主路径的设置与左右对比还原测试弹层，主要文案已双语化
- `AIGeneratorDialog.tsx`：未接入主路径的 AI 生成弹层，主要文案已双语化
- `WireframeBlock.tsx`：七类活跃组件低保真骨架渲染，预览态由根节点承接交互，并提供普通选中与放置锁定高亮两类强调样式钩子；待放置时可切到纯展示态；块内名称空值不提交
