1. **地位**：编辑器共用辅助能力

2. **逻辑**：集中固定设备预设、七类活跃组件常量、旧组件导入兼容 metadata、双语词典、浏览器语言检测、contain 缩放、画布缩放文案、布局与导入导出辅助、AI 请求与 iframe 预览构造；组件归一化只做最小尺寸约束，复制辅助支持可控偏移，组件创建/重命名支持最小防重；用户创建路径只暴露 layout/content/input/navigation/feedback/media/commerce，不暴露 modal 或旧细分组件；导出 JSON 时补 instruction、_instructions、clipped 语义，并追加手绘容差与禁 emoji 备注，把内部 description 映射为导出 info，本地存储仅持久化单一工作区与 wireframe purpose。

3. **约束**：仅保留复用度明确的纯函数；不要堆叠抽象层。

## 成员清单

- `constants.ts`：固定设备预设、七类活跃组件、旧组件兼容 metadata 与待放置阈值常量
- `i18n.ts`：中英词典、七类组件标签、浏览器语言检测、画布缩放、命名辅助与操作 toast 文案
- `project.ts`：项目创建、组件布局、contain 缩放、放置规则、JSON 导入导出与 clipped/手绘容差/禁 emoji 备注派生
- `storage.ts`：按单一工作区格式写入 localStorage，并做防抖保存
- `ai.ts`：OpenAI 兼容请求、按 locale 切换提示词，并让还原更偏向结构语义而非机械坐标
